# goravel-vue-admin

### 技术栈
vue3 + element-plus  + tailwindcss + vite

### 安装 yarn
```go
npm install -g yarn
yarn -v
```

### 启动服务

```go
// 安装依赖
yarn install
// 启动服务
yarn dev
```

访问 http://localhost:9000/cms 进入后台登录页面，默认系统管理员账号/密码为 `admin`/`123456`
